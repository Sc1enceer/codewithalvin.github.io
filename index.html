Some Example of My Recent Projects

1. PhD Research Project: Balancing Cost and Comfort in HVAC Control with Human-Centric Deep Reinforcement Learning

My research focused on developing an innovative control strategy for HVAC systems, addressing the challenges of balancing energy efficiency with user comfort. Leveraging Deep Reinforcement Learning (DRL) and integrating occupant feedback, the model optimized HVAC operations by dynamically adapting to environmental uncertainties, room occupancy, and personalized user preferences.

Research Challenges and Problem-Solving Approaches
  • Uncertainty Estimation: Addressed dynamic factors such as electricity prices and user comfort preferences by designing predictive models using multi-step LSTM networks and comparing with alternatives like ARIMA and Decision Trees.
  • Personalized User Preferences: Developed innovative reward functions incorporating user-specific weightings for cost and thermal comfort using feedback mechanisms such as inverse reinforcement learning.
  • Feedback Mechanisms: Explored Human-In-The-Loop Reinforcement Learning (HITL-RL) to integrate occupant feedback in real-time through interactive interfaces, improving agent adaptability.
  • Simulation and Evaluation: Used agent-based modeling to simulate interactions and validate learning capability under realistic conditions before deployment.
Methodologies and Key Innovations
  • Reinforcement Learning Algorithms: Implemented value-based (Deep Q Network) and policy-based (Deep Deterministic Policy Gradient) models to dynamically adapt HVAC controls, achieving near-optimal performance.
  • Data Processing: Designed preprocessing pipelines to enhance data reliability for DRL training, including normalization and feature extraction for sequential data.
  • Interactive Learning: Employed advanced HITL approaches for feedback-driven learning, leveraging techniques like policy shaping and reward restructuring.
Outputs and Achievements
  • Successfully developed a novel DRL-based HVAC control model that incorporates occupancy features, enabling energy-efficient operations without compromising user comfort.
  • Created predictive models for electricity price trends using LSTM networks, demonstrating superior performance in handling long sequences of data.
  • Advanced the state-of-the-art in HVAC optimization by integrating real-time user feedback and personalizing control policies to adapt to dynamic conditions.

Details and Goals

  • Optimization Studies: Conducted in-depth analyses to determine optimal locations for EV charging stations using tools like Geographic Information Systems (GIS) and optimization algorithms. Addressed the closure of petrol stations to optimize resource coverage.

4. Implementation of Deep Convolutional Generative Adversarial Network (DCGAN) for Architectural Innovation

This project explored and implemented Generative Adversarial Networks (GANs) to produce high-quality synthetic architectural images. A custom dataset of 10,000 images was created, and a DCGAN architecture was designed and compared with variants like WGAN-CP and WGAN-GP. Despite challenges such as limited computing resources and the global pandemic, the project demonstrated the effectiveness of DCGAN in generating realistic architectural designs, achieving the highest Inception Score among the models tested.

Details and Goals
  • Created a custom architectural image dataset using a web crawler and preprocessing pipeline for GAN training.
  • Conducted research on GAN techniques to address mode collapse and non-convergence issues.
  • Developed and trained a DCGAN model on Google Colab, incorporating gradient penalties and optimizing for JS divergence.
  • Evaluated models on speed, image quality, and metrics like the Inception Score, with DCGAN achieving superior results.
Methodologies
  • Used deep learning frameworks like PyTorch for model development and experimentation.
  • Applied techniques such as data normalization, batch processing, and adaptive optimizers for efficient training.
  • Incorporated quantitative metrics like the Inception Score and visual analysis for robust model evaluation.
